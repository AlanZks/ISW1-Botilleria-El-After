# ISW1-Botilleria-El-After

Repositorio correspondiente al proyecto del curso Ingeniería de Software I de la Universidad Finis Terrae.

## Proyecto

Sistema de Gestión de Ventas e Inventario para Botillería El After.

## Descripción

El proyecto tiene como objetivo desarrollar un sistema que permita optimizar la gestión interna de la Botillería El After mediante el registro digital de ventas, control de inventario y generación de reportes básicos de gestión.
